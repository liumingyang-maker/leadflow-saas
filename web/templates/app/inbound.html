② 方式一（推荐）：嵌入一段代码，访客留在你网站

把下面代码贴到你网站每个页面的 </body> 之前， 并给你的询盘表单 <form> 标签加一个 data-leadflow 属性即可。 不影响你表单原本的提交行为。 

<script>
(function(){
  var ENDPOINT = "{{ endpoint }}";
  document.querySelectorAll('form[data-leadflow]').forEach(function(form){
    form.addEventListener('submit', function(){
      var obj = {}; new FormData(form).forEach(function(v,k){ obj[k]=v; });
      fetch(ENDPOINT, {method:'POST', keepalive:true,
        headers:{'Content-Type':'application/json'}, body:JSON.stringify(obj)
      }).catch(function(){});
    });
  });
})();
</script>

③ 表单字段命名（让数据对上号）

你表单里输入框的 name 用下面这些词，系统就能自动识别（大小写都行，邮箱或电话至少有一个）：

{% for label, names in [ ('邮箱','email / mail / e-mail'), ('姓名','name / contact_name / fullname'), ('公司','company / company_name / organization'), ('电话/WhatsApp','phone / tel / mobile / whatsapp'), ('国家','country'), ('留言内容','message / comment / inquiry'), ('网址','website / url'), ] %} {% endfor %}
对应客户字段表单 name 可用
{{ label }}{{ names }}

④ 方式二：没有现成表单？直接用这个

把这段贴到网页里就是一个能用的询盘表单（适合不会写代码的情况）。

<form data-leadflow>
  <input name="name" placeholder="Your Name">
  <input name="email" placeholder="Email" required>
  <input name="company" placeholder="Company">
  <input name="phone" placeholder="Phone / WhatsApp">
  <input name="country" placeholder="Country">
  <textarea name="message" placeholder="Your inquiry"></textarea>
  <button type="submit">Send Inquiry</button>
</form>
