3. 密钥走环境变量；本地放 `.env`（已 gitignore）；`DATA_DIR` 指数据目录（云上挂 `/data` 持久卷）
4. **采集器统一模式**：`class XxxCollector: fetch_all(mock=False)->list[dict]`；
   在 `web/app.py` 的 `run_bg()` 加 `if "xxx" in channels` 块；`collect.html` 加卡片（checkbox id=`ch-xxx`，渠道名 = 去掉 `ch-` 前缀）
5. 入库统一：`DataCleaner().run(raw_list, source="xxx", db_path=tenant_ctx.get_db_path(tid))`
6. 新功能多为"原生等价实现"而非装 GitHub 原版（轻依赖、即装即用、境内可跑）
7. 数据隔离：每租户 `tenants/{tid}/`（leads.db + config.json + email_templates.json）；
   全局 `admin.db`（租户账号 + inbound_tokens + email_tracking + followups 表）
